Q-1: What is DOM and what is the purpose of DOM?

DOM:
The Document Object Model (DOM) is a programming interface for web documents that represents the page so that programs can change the document structure, style, and content.

Purpose of DOM
The purpose of the DOM is to provide a platform-neutral programming interface for web documents that can be used by various programming languages such as JavaScript, Python, and Ruby. The DOM provides a set of methods and properties that enable developers to access and manipulate the content of web pages dynamically at runtime.

Q-3: What is event bubble?

When an event is triggered on an element, such as a click event on a button, the event is first executed on the target element, and then the event bubbles up to the parent element, and then to its parent, and so on until it reaches the top of the DOM tree.

Q-4: What is a callback function and why will you use it?

A callback function is a function that is passed as an argument to another function and is executed by that function at a specific point in time or in response to a particular event or condition. In other words, a callback function is a function that is "called back" by another function.
